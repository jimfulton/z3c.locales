This package provides the translation for the `z3c` packages which are
using the shared `z3c` domain.

.. contents::

==========
 Contents
==========

This package provides translations for:

- z3c.authviewlet
- z3c.contents
- z3c.layer.pagelet
- z3c.table
- z3c.wizard

=====
Usage
=====

Include the ``configure.zcml`` of this package in your project.
